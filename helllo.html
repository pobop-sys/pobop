Backrooms — Quick Level Guide

Short, copy-friendly descriptions of the levels implemented in your prototype: what to expect, transitions, and console tricks.

Level 0 — Lobby

The starting 'lobby' area — maze-like with denser corridors.

Level 0 acts as the primary hub in the prototype. It uses a denser, tightened maze layout so corridors feel cramped and maze-like.

  • Spawn: Often you begin here.
  • Exit: Reaching the exit normally moves you to Level 1.
  • Special: A red "Hell" door can appear here under certain conditions and teleports you to Level 666.
How to trigger the red door: the reverse-Konami input or the dev console hell command will spawn a red door somewhere in Level 0.

Level 1 — Corridor

Long, narrow corridors — dimmer, with a clear exit tile.

Level 1 is a simpler corridor map (open-ish, but long). In normal gameplay reaching the exit takes you deeper to Level 2 (the storage level).

  • Spawn: Teleported or placed based on safe spawn logic.
  • Exit behavior: When you step on the exit tile while in Level 1, you are moved to Level 2.
  • Visuals: Darker ceiling/floor shades are used to set mood.

Level 2 — Storage

A new 'storage' level added to the flow: Level 1 → Level 2 → Level 0 loop.

Level 2 is intended as a deeper storage area (the file uses level_2_maze). In the updated flow, stepping on the exit in Level 2 returns you to the Lobby (Level 0), creating a simple loop.

  • Purpose: Acts as a mid-tier level between the corridor and the lobby.
  • Exit behavior: Exit from Level 2 returns the player to Level 0.
  • Notes: This level is intentionally simple by default; you can expand the maze or add unique visuals later.

Level 666 — Hell

A hostile, hellish area with a randomly generated maze.

Level 666 is generated with a random carve algorithm so every spawn is chaotic and maze-like. If you reach the exit in this level you are sent to the corrupted lobby (Level -0).

  • Spawn: Entered via the Hell door (red door spawned in Level 0) or developer teleport.
  • Exit: Moves player into Level -0 (corrupted lobby) to continue the eerie sequence.
  • Visuals & sound: More aggressive audio & warmer red-toned walls for atmosphere.

Level -0 — Corrupted Lobby

A corrupted, glitching version of the lobby with broken audio/visuals.

Level -0 (internally LEVEL_NEG0) is a distorted variant of the lobby. It includes glitch overlays, tinting, and an intentionally broken humming sound effect that fades, cuts out, and stutters.

  • Behavior: Movement can stutter (random skips) to convey corruption.
  • Exit: Leaving Level -0 takes you into Level 1 (the corridor) to continue the loop.
  • Glitches: Random visual artifacts appear on a timed interval; an audio glitch plays when a glitch starts.

Dev Console & Useful Commands

Console and secret controls supported in the prototype.

The prototype includes a dev console (openable via a Konami-like key sequence). Several quick commands help testing and teleporting.

  • Open console: Enter the Konami sequence (up, up, down, down, left, right, left, right) to open the console.
  • Commands:
    • help — show available commands
    • levels — list available levels (0,1,2,666,-0)
    • goto <n> — teleport to a level (example: goto 2)
    • gt / teleport — click overlay to teleport
    • hell — spawn the Hell door (red door) in Level 0
    • arrow — toggle HUD guidance arrow
  • Reverse konami: An alternate Konami triggers the Hell door automatically when pressed in Level 0.
Quick dev tip: use goto 2 to test Level 2, and step on Level 2's exit to verify you return to Level 0.

Example console usage

> levels
Levels: 0=lobby, 1=corridor, 2=storage, 666=hell, -0=corrupted-lobby
